Сборка DIRIT start
---------------------------------------------------------
Страница проекта http://dirit.ru/sborka-sayta-dirit-start
---------------------------------------------------------

Инструкция по установке:

1. Распаковать содержимое архива site_folder.tgz в корневую папку сайта

2. Распаковать содержимое архива main.sql.tgz, извлечённый файл main.sql экспортировать в предварительно созданную для сайта базу данных

3. Отредактировать файл sql.txt:
3.1 заменить все вхождения --email-- на email администратора (данный email будет использован для генерации пароля администратора)
3.1 заменить --str_len-- на число символов в email администратора (например, для example@yandex.ru - 17)

4. Переименовать файл sql.txt в sql.sql

5.файл sql.sql экспортировать в предварительно созданную для сайта базу данных

6. Отредактировать файл php.txt:
6.1 заменить --hash_salt-- на произвольную строку произвольной длины, например - 50 символов
6.2 заменить --username-- на имя пользователя базы данных
6.3 заменить --password-- на пароль пользователя базы данных
6.4 заменить --database-- на название базы данных

7. Переименовать файл php.txt в settings.php

8. Переместить файл settings.php в папку /sites/default (внутри корневой папки сайта)

9. Открыть свой сайт на странице восстановления пароля ( http://dynurl.net/user/password ), ввести email из пункта 3.1, на указанный email придёт письмо с ссылкой для генерации нового пароля администратора
